Git

Базовые операции

  • git init — создание нового репозитория
  • git clone ссылка — клонирование репозитория
  • git remote add origin ссылка — связь с удаленным репозиторием

Работа с изменениями

  • git status — проверка статуса файлов
  • git add файл — добавление файла в индекс
  • git add . — добавление всех файлов
  • git commit -m "сообщение" — создание коммита
  • git log — просмотр истории коммитов

Синхронизация

  • git pull — получение изменений
  • git push — отправка изменений
  • git push -u origin ветка — первая отправка ветки

Работа с ветками

  • git branch — список веток
  • git switch ветка — переключение на ветку
  • git checkout -b ветка — создание новой ветки
  • git merge ветка — слияние веток

Отмена изменений

  • git restore файл — отмена изменений файла
  • git reset --hard — полный откат изменений
  • git stash — сохранение изменений

Дополнительные команды

  • git diff — сравнение версий
  • git show хеш — детали коммита
  • git clean — очистка файлов
